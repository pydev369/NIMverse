# Backend

This directory contains the backend application code for the agentic application with NVIDIA NIM.

## Structure

- `src/` - Main source code
- `tests/` - Test files
- `Dockerfile` - Docker configuration
- `requirements.txt` - Python dependencies

## Components

- Main application server
- Agent orchestrator
- Retrieval tools
- Summarization tools
- Analytics tools
- Workflow management
- Memory store
- Utility functions
