# Frontend

This directory contains the frontend application code for the agentic application with NVIDIA NIM.

## Structure

- `src/` - Main source code
- `components/` - React components
- `hooks/` - Custom React hooks
- `styles/` - CSS styles
- `lib/` - Library functions
- `Dockerfile` - Docker configuration
- `package.json` - Node.js dependencies
- `vite.config.js` - Vite configuration

## Components

- Main application UI
- Agent dashboard
- Cost monitoring display
- Chat panel interface
- API hooks
- Utility functions
